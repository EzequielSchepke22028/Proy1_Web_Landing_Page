# MarketEze - E-commerce Platform (Full-Stack)

Proyecto desarrollado por Ezequiel Schepke

## 🚀 Tecnologías Utilizadas
- **Backend**: Python 3.12, FastAPI, SQLAlchemy, PostgreSQL, JWT para autenticación.
- **Frontend**: React 18, Tailwind CSS, React Router DOM, Axios.
- **Infraestructura**: Docker Desktop, Virtual environments (venv).

## 🛠️ Funcionalidades Implementadas
- [x] Registro de usuarios con hashing de contraseñas (Bcrypt).
- [x] Login con generación de Access Tokens (JWT).
- [x] Middleware de CORS configurado para comunicación segura.
- [x] Frontend con Context API para manejo de estado global de usuario.
- [x] Rutas protegidas y diseño responsive con Tailwind.
